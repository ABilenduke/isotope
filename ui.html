How it Works

This plugin allows you to manage Figma variables using JSON files. You can import, export, and delete variables in bulk.

Creates & Updates

The plugin uses an "Upsert" strategy:

  • Create: If a collection, mode, or variable in your JSON doesn't exist in Figma, it will be created.
  • Update: If it already exists (matched by name), its value and properties will be updated.
  • Ignore: Existing variables in Figma that are NOT in your JSON are left untouched.

JSON Structure

Your JSON file should follow this hierarchy:

{
  "Collection Name": {
    "Mode Name": {
      "Group": {
        "Variable": { 
          "$type": "color", 
          "$value": "#FF0000" 
        }
      }
    }
  }
}

Supported Types

  • color: Hex codes (e.g., "#FF0000")
  • number: Numbers (e.g., 16, 0.5)
  • boolean: true / false
  • string: Text values

Aliases

To reference another variable, use the syntax:

"{Collection.Group.Variable}"

Example: "{Brand.Color.Primary}"
